请帮我写一个Python函数，计算斐波那契数列的第n项。

以下是使用动态规划高效计算斐波那契数列的Python函数：

def fibonacci(n):
    if n <= 0:
        return "输入必须为正整数"
    elif n == 1:
        return 0
    elif n == 2:
        return 1
    else:
        a, b = 0, 1
        for _ in range(2, n):
            a, b = b, a + b
        return b

# 测试
print(fibonacci(10))  # 输出：34

这个实现的时间复杂度为O(n)，空间复杂度为O(1)，非常高效。
